# Dataflow: Qwik Start - Python (GSP207)

## Objective
To learn how to build and run a simple Apache Beam pipeline using Google Cloud Dataflow, including both local execution and remote execution on Google Cloud.

## Key Steps Performed
1. Enabled the Dataflow API.
2. Created a Cloud Storage bucket to store pipeline output.
3. Set up a Docker environment with Python 3.
4. Installed Apache Beam SDK for Python using pip.
5. Ran a WordCount pipeline locally using DirectRunner.
6. Verified output files from local execution.
7. Configured BUCKET environment variable for Cloud Storage.
8. Executed WordCount pipeline remotely using DataflowRunner.
9. Monitored job progress in Dataflow console.
10. Verified successful pipeline completion.
11. Checked output results stored in Cloud Storage bucket.

## Tools / Services Used
- Google Cloud Dataflow
- Apache Beam (Python SDK)
- Cloud Storage
- Cloud Shell
- Docker
- gcloud CLI
- Python 3

## Results
Successfully executed a WordCount pipeline using Apache Beam both locally and on Google Cloud Dataflow. The job was executed remotely and results were stored in a Cloud Storage bucket.
