# Cloud Run Functions: Qwik Start - Command Line

## Objective

To learn how to create, deploy, and test a serverless Cloud Run Function using the Google Cloud command-line interface.

## Key Steps Performed

1. Configured the default Cloud Run region.
2. Created a Node.js Cloud Run Function project.
3. Developed a Pub/Sub-triggered function in index.js.
4. Created the package.json configuration file.
5. Installed required dependencies using npm.
6. Deployed the function using gcloud functions deploy.
7. Configured a Pub/Sub topic trigger.
8. Published a test message to the Pub/Sub topic.
9. Verified successful execution through Cloud Logging.
10. Reviewed function logs using the gcloud command line.

## Tools/Services Used

* Cloud Run Functions (Gen2)
* Cloud Shell
* gcloud CLI
* Pub/Sub
* Cloud Logging
* Node.js
* npm

## Results

Successfully deployed a Cloud Run Function that was triggered by a Pub/Sub message. Verified function execution and log generation through Cloud Logging.
