# Create a Virtual Machine

## Objective
To learn how to create and manage virtual machine instances using Compute Engine through both the Cloud Console and the gcloud command-line tool.

## Key Steps Performed
1. Created a VM instance (gcelab) using the Cloud Console.
2. Connected to the VM using SSH.
3. Installed the NGINX web server.
4. Verified the NGINX welcome page through the external IP address.
5. Created a second VM instance (gcelab2) using gcloud commands.
6. Connected to the second VM using SSH.

## Tools/Services Used
- Google Compute Engine
- Cloud Shell
- gcloud CLI
- NGINX
- SSH

## Results
Successfully created two VM instances and deployed an NGINX web server that was accessible through a web browser.
